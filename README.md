# Unusual-Minigolf

Source code for Unusual Minigolf game made in AutoHotkey.

Depends on:
- AutoHotkey v2.0.3
- raylib 4.5-dev (latest version should work too)

raylib should be compiled as a shared library and placed in the `dll` directory.
